# Offline-First Operation

Velvet is designed to remain useful and safe without continuous internet access.

## Local Responsibilities

The following belong locally:

- identity and continuity
- Court policy
- safety gates
- approved executors
- core receipts
- body and vehicle observation
- primary scenes and local interaction
- essential language and reasoning

## Cloud-Optional Services

Cloud services may assist with research, drafting, updates, or language range, but their absence must not erase identity, authority, or core operation.

## Update Boundary

Updates should be treated as untrusted until verified by the local installation. A network download is not an accepted update merely because transport succeeded.

## Degraded Mode

When internet access is unavailable, Velvet should report the missing capability clearly and continue with approved local functions.

## Principle

```text
cloud may assist
local system decides
physical body remains locally governed
```
